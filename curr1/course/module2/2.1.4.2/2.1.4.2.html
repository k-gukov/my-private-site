Справочное руководство по командам CISCO IOS

Справочное руководство по командам CISCO IOS — это набор онлайн-документации, которая подробно описывает команды IOS, используемые в устройствах Cisco. Справочное руководство — основной источник информации для определённой команды IOS, точно так же, как словарь — основной источник для получения информации о конкретном слове.

Справочное руководство — основной источник, которым пользуются разработчики сетей для проверки различных характеристик заданной команды IOS. Ниже перечислены некоторые из наиболее распространенных характеристик:

  • Синтаксис — наиболее подробная версия синтаксиса команд из существующих

  • По умолчанию — способ реализации команды на устройстве с конфигурацией по умолчанию

  • Режим — режим конфигурации на устройстве для ввода команды

  • История — описания реализации команд в соответствующей версии IOS

  • Инструкции по применению — рекомендации по реализации команд

  • Примеры — полезные примеры, демонстрирующие распространённые случаи использования команды

Для перехода к Справочному руководству и поиска определённой команды выполните следующие действия:

Шаг 1. Перейдите по адресу www.cisco.com.

Шаг 2. Перейдите во вкладку Поддержка(Support)

Шаг 3. Выберите Сетевое программное обеспечение (IOS и NX-OS).

Шаг 4. Выберите 15.2M&T (в качестве примера).

Шаг 5. Выберите Справочные руководства.

Шаг 6. Выберите Справочныеруководства

Шаг 7. Выберите конкретную технологию, содержащую команду, на которую вы ссылаетесь.

Шаг 8. Выберите ссылку слева, соответствующую алфавитному указателю команды, на которую вы ссылаетесь.

Шаг 9. Нажмите ссылку этой команды.

Например, команду description можно найти в Справочном руководстве аппаратных компонентов и интерфейса Cisco IOS под ссылкой алфавитного порядка от D до E.

Примечание. Полные версии PDF Справочных руководств для конкретной технологии можно загрузить по ссылкам на странице, которую вы увидите после завершения шага 7.
