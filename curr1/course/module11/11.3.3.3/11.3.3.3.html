Просмотр настроек коммутатора с помощью команды «show version»

Команда show version на коммутаторе отображает сведения о текущей загруженной версии программного обеспечения, а также информацию об аппаратном обеспечении и устройстве. С помощью этой команды отображается, например, следующая информация.

  • Версия программы — версия программного обеспечения IOS

  • Версия программы начальной загрузки — версия программы начальной загрузки

  • Время работы системы — время с момента последней перезагрузки

  • Сведения о перезагрузке системы — способ перезапуска (например, цикл включения-выключения, сбой системы)

  • Имя образа программы — имя файла образа IOS

  • Платформа коммутации и тип процессора — номер модели и тип процессора

  • Тип памяти (общая/основная) — основная ОЗУ процессора и буферизация ввода-вывода общего пакета.

  • Аппаратные интерфейсы — доступные интерфейсы на коммутаторе

  • Регистр конфигурации — задаёт спецификации начальной загрузки, настройку скорости консоли и другие связанные параметры.
