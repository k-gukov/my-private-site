Параметры команды «ipconfig»

Как показано на рисунке 1, чтобы просмотреть IP-адрес шлюза по умолчанию для узла, можно запустить команду ipconfig в командной строке компьютера под управлением Windows.

Для просмотра MAC-адреса компьютера используется инструмент ipconfig /all. Обратите внимание, что на рисунке 2 MAC-адрес компьютера отображается теперь вместе с подробными сведениями об адресации устройства 3-го уровня. Попробуйте использовать эту команду.

Кроме того, производителя сетевого интерфейса для компьютера можно определить по части OUI MAC-адреса. Эти сведения можно найти в Интернете.

Клиентская служба DNS на ПК с операционной системой Windows оптимизирует производительность разрешения имен DNS за счёт хранения ранее разрешённых имён в памяти. На ПК с операционной системой Windows команда ipconfig /displaydns выводит на экран все кэшированные записи DNS.
