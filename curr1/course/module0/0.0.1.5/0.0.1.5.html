Освоение предмета через практический опыт

По окончании занятия необходимо проверить свои знания при помощи интерактивных материалов. Если требуется запомнить новые команды, то сначала их следует отработать при помощи средства по проверке синтаксиса, а уже потом использовать для настройки сетей и устранения неполадок в них с помощью программы Packet Tracer, являющейся инструментом моделирования сетей в программе Сетевой академии. Далее вы будете выполнять лабораторные работы на реальном оборудовании в вашем классе или с помощью удаленного доступа к оборудованию через Интернет.

При помощи программы Packet Tracer можно также отработать навыки, создавая свои упражнения или принимая участие в групповых играх с другими учащимися. Практические упражнения по оценке и интеграции навыков программы Packet Tracer позволяют наиболее эффективно продемонстрировать полученные навыки. Их полезно использовать для отработки материалов глав, при подготовке к зачётам и выпускным экзаменам.
