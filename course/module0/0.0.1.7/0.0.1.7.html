Инженерно-технические журналы

Специалисты в области сетевых технологий нередко ведут инженерно-технические журналы, в которые записывают свои наблюдения и опыт (например, способы использования протоколов и команд). В процессе ведения инженерно-технических журналов создаются справочные материалы, которые можно использовать в работе ИКТ. Ведение записей наряду с чтением, наблюдением и практикой — один из способов закрепить полученные знания.

Пример записи для внедрения технологии может включать необходимые для ее настройки команды программного обеспечения, назначение команд, переменные и схему топологии с указанием контекста использования команд.

Для получения доступа к этому курсу войдите в систему netacad.com.
