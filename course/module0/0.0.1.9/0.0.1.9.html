Создание собственного мира

Программу Packet Tracer можно также использовать для создания собственных экспериментов и сетевых сценариев. Надеемся, что впоследствии вы начнете использовать Packet Tracer не только для выполнения предложенных упражнений, но и для самостоятельной работы по обучению, использованию и применению приобретенных навыков.

Материалы учебного курса содержат упражнения с использованием программы Packet Tracer, которые можно открыть на компьютерах с операционными системами Windows®, если на них установлен Packet Tracer. Интеграция также может работать в других операционных системах, использующих эмуляцию Windows.

Для получения доступа к этому курсу войдите в систему netacad.com.
