Конфигурация маршрутизируемого порта на коммутаторе уровня 3

Порт коммутатора можно настроить таким образом, чтобы он функционировал в качестве маршрутизируемого порта уровня 3 и действовал в режиме стандартного интерфейса маршрутизатора. В частности, порт с маршрутизацией:

  • не связан с определённой виртуальной локальной сетью (VLAN);

  • может быть настроен для работы с протоколом маршрутизации уровня 3;

  • является только интерфейсом уровня 3 и не поддерживает протокол уровня 2.

Настройте маршрутизируемые порты, переведя интерфейс в режим уровня 3 с помощью команды настройки интерфейса no switchport. Затем назначьте IP-адрес порту. Вот и всё!
