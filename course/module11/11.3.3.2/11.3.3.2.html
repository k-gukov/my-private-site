Просмотр настроек маршрутизатора с помощью команды «show version»

После успешной загрузки файла загрузочной конфигурации и загрузки маршрутизатора можно воспользоваться командой show version для проверки основных аппаратных и программных компонентов, которые используются в процессе загрузки, а также для устранения неполадок. После выполнения команды show version выводятся следующие выходные данные.

  • используемая версия программы Cisco IOS;

  • версия программы начальной загрузки системы, которая хранится в памяти ПЗУ и которая использовалась для первоначальной загрузки маршрутизатора;

  • полное имя файла образа Cisco IOS и его местонахождение, найденное программой начальной загрузки;

  • тип центрального процессора маршрутизатора и объём ОЗУ. При обновлении программы Cisco IOS может потребоваться увеличение объёма ОЗУ.

  • Количество и тип физических интерфейсов маршрутизатора.

  • Объём NVRAM (энергонезависимого ОЗУ). NVRAM служит для хранения файла начальной конфигурации.

  • Объём флеш памяти маршрутизатора. При обновлении программы Cisco IOS может потребоваться увеличение объёма флеш памяти.

  • Текущее установленное значение конфигурационного регистра программы — шестнадцатеричное.

На рисунке нажмите кнопку «Воспроизведение», чтобы просмотреть анимированное представление по определению этих функций в выходных данных команды «show version».

Конфигурационный регистр определяет процедуру загрузки маршрутизатора. Например, заводская настройка по умолчанию для конфигурационного регистра устанавливает значение 0x2102. Такое значение указывает, что маршрутизатор будет пытаться загрузить программу Cisco IOS из флеш памяти, а файл загрузочной конфигурации — из NVRAM. Значение конфигурационного регистра можно изменить, изменив тем самым место, где маршрутизатор будет искать образ Cisco IOS и файл загрузочной конфигурации в процессе загрузки. Если в скобках указано второе значение, то это — значение конфигурационного регистра, которое будет использовано при следующей перезагрузке маршрутизатора.

Нажмите значок «Примечание» в нижнем правом углу рисунка, чтобы просмотреть дополнительные сведения о конфигурационном регистре.
