Повторное рассмотрение наиболее распространённых команд типа «show»

Команды Cisco IOS CLI show служат для отображения соответствующих данных о конфигурации и работе устройства.

Специалисты по обслуживанию сети широко пользуются командами show для просмотра файлов конфигурации, проверки состояния интерфейсов и процессов на устройствах, а также для контроля рабочего состояния устройств. Команды show можно использовать в том случае, если устройство настроено с помощью CLI или специалистом Cisco по конфигурации.

С помощью команды show можно отобразить состояние практически любого процесса или функции маршрутизатора. К наиболее известным командам «show» относятся:

  • show running-config (Рисунок 1)

  • show interfaces (Рисунок 2)

  • show arp (Рис. 3)

  • show ip route (Рис. 4)

  • show protocols (Рисунок 5)

  • show version (Рисунок 6)

Чтобы просмотреть дополнительные сведения о командах show, используйте кнопки на рисунке.
